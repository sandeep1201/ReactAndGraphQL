1.2.0

Version 1.0

Version 1.0 denotes the initial release of the WWP software. This version only impliments PIN searching and direct navigation to the Time Limit functionality. Stable releases will be released to production, development releases are only available to developers or users who perform testing.

1.2 - Time Limits Update, WPASS End-to-End UAT Release

2/23/2018

Added Features / Changes

  • For cases transferred to a new W-2 agency, the system will now recognize the new Work Programs office assignment and FEPs in the new W-2 agency will be able to make month edits and extension entries.
  • Months and extensions added in WWP will updated in CWW. FEPs will be able to enter placements.
  • Extension decisions will update real-time in WWP.
  • For Time Limits, two new tabs have been added to the WWP Dashboard page. The first tab displays the users recently accessed PINS. Up to 10 PINS will display. The second tab is where users will now do PIN searches.

1.1.5 - Sprint 26 Release

12/18/2017

Known Issues

  • None

Bug Fixes

  • DCF 6: In certain scenarios, users were unable to add multiple OTF months. This is resolved.
  • DCF 13: Some two-parent assistance groups did not display both parents on the Time Limit page. Both parents now display.
  • DCF 42: Users can no longer add months for minors.
  • DCF 49: Extension reasons now align with policy and only allowed reasons display.
  • DCF 54: Extensions entered into WWP now display real time.
  • DCF 64: Some W-2 assistance groups were closing for “expired clocks” even though there were months remaining. This is resolved.
  • Additionally, users will now be able to make multiple month edits without having to refresh the application between edits.
