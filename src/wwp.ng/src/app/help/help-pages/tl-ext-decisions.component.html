Extension Decisions

Description: This page displays any extensions that have been entered for the individual. Use this page to view past extension decisions, end current extensions, or add new decisions for the same extension timeframe.

Extension Decisions Extension Details
Page Fields Page Field Descriptions
Extension Decisions
Filter By Type

This dropdown menu allows the FEP to view extensions by time limit type. Only time limit types for which the individual has received an extension are selectable.

Show/Hide Deleted Decisions

This button allows the FEP to show or hide extensions that were entered in error and deleted prior to starting.
Decision

This displays whether an extension has been approved or denied, and the applicable time limit type.
Decision Date

This information is system-generated based on the date the FEP enters the extension decision.
Begin Month

This field displays the month and year the extension is scheduled to begin.
End Month

This field displays the month and year the extension is scheduled to end.
Reason

This field displays the approval or denial reason for the extension.
Icons

This will end an extension that has already started at the end of the current month.

This will delete an extension that was entered in error that has not yet started. This icon will not display if the extension has already started.

This will allow the FEP to add a new extension decision for the same extension timeframe as a previously entered decision. This icon will not display if the extension has already started.
Add/Deny Extension

This button opens the Add Extension Decision window and allows the FEP to enter a new extension decision. This button is disabled if the individual is not currently eligible for a new extension.
Extension Details
Decision Tab

This tab shows the Extension Decision and Month/Year of the Decision Date. Multiple tabs will display if multiple decisions have been entered for the same timeframe.
Details

The section below the tab displays details regarding the decision, including the decision, the applicable time limit type, the decision date, begin and end months, the reason for the extension, and any details that were entered when the extension was entered. This section also displays the last editor and the date and time the decision was edited.
Hide/Show Deleted Decision

This button allows the FEP to only view Approvals for the extension timeframe, or to view all decisions entered for the extension timeframe.
Add Extension Decision

This button allows the FEP to enter a new decision for the same extension timeframe. This button will be disabled if the extension has already started.
Delete Decision

This button allows the FEP to delete an extension decision that was entered in error. This button is disabled if the extension has already started.
